# Development

## Install

```bash
pnpm install
```

If Corepack is broken in this sandbox, run:

```bash
./scripts/ensure-pnpm.sh install
```

## Build

```bash
pnpm build
```

## Test

```bash
pnpm test
```

## Lint

```bash
pnpm lint
```

## Typecheck

```bash
pnpm typecheck
```

## CLI help

```bash
pnpm mediaforge --help
```

## Doctor

```bash
pnpm run doctor
```
