# Feature Inventory

| Feature | Customer-facing description | Evidence | Status | Notes |
| ------- | --------------------------- | -------- | ------ | ----- |
| Local source ingestion | Imports local media with sidecar transcript files. | `packages/source-ingestion/src/index.ts`, `packages/pipeline/src/index.ts` | Available | YouTube and TikTok adapters inspect URLs but require manual transcript acquisition. |
| Transcript generation and normalization | Creates transcripts from OpenAI-compatible APIs or whisper.cpp and normalizes word/segment timing. | `packages/transcription/src/index.ts`, `apps/cli/src/index.ts` | Available | Local whisper.cpp is optional. |
| Transcript cleaning | Removes filler, repeated text, and production contamination before rewrite. | `packages/transcript-cleaning/src/index.ts`, `packages/story-localization/src/source-cleaning.ts` | Available | Conservative and OpenAI-compatible paths exist. |
| Full story rewriting | Produces canonical English full narration and localized full stories. | `apps/cli/src/story-full-rewrite-command.ts`, `packages/story-localization/src/story-localization.service.ts` | Available | Uses structured OpenAI-compatible responses. |
| Short-form rewriting | Creates short adaptations with beat selection, duration targets, and parent-story validation. | `apps/cli/src/story-short-rewrite-command.ts`, `packages/story-localization/src/short-adaptation-contract.ts` | Available | Supports English and localized short outputs. |
| Supported story languages | English source plus German, Spanish, French, and Portuguese localization profiles. | `packages/story-localization/src/story-localization.types.ts`, `packages/story-localization/src/language-profiles.ts` | Available | Episode media commands are narrower in places: `en`, `de`, `es`, `fr`. |
| Canonical facts | Extracts characters, setting, objects, events, written messages, threat, reveal, and consequence. | `packages/story-localization/src/canonical-facts.service.ts`, `packages/story-localization/src/story-production.ts` | Available | Heuristic extraction, later validated by contracts. |
| StoryIR | Normalizes entities, facts, genre, fictionality, narrative mode, rules, objects, and chronology. | `packages/story-localization/src/story-artifact-model.ts` | Available | Customer-facing term: story intelligence model. |
| Story bible | Builds a compact cast, premise, rules, scene order, key objects, and protected story elements. | `packages/story-localization/src/story-production.ts` | Available | Used as continuity and rewrite context. |
| Character maps | Builds person entities, cast, relationships, aliases, deterministic rename maps, and visual registry entries. | `packages/story-localization/src/story-artifact-model.ts`, `packages/story-localization/src/character-rename.service.ts`, `apps/cli/src/episode-commands.ts` | Partially available | Rich enough for episode continuity; not a full cross-series identity graph. |
| Character appearance profiles | Stores physical description, face, hair, build, wardrobe, carried objects, traits, and reference status. | `packages/image-generation/src/episode-image-pipeline.ts`, `apps/cli/src/episode-commands.ts` | Available | Profiles are generated from source facts with heuristics. |
| Relationships | Captures character relationship text in canonical facts, StoryIR, story bible, and rename map roles. | `packages/story-localization/src/story-localization.types.ts`, `packages/story-localization/src/story-artifact-model.ts` | Partially available | No graph analytics beyond textual relationship fields. |
| Scene participation | Tracks scene character usage, unresolved recurring mentions, and character IDs in scene image jobs. | `packages/image-generation/src/episode-image-pipeline.ts`, `packages/image-generation/src/image-batch.schemas.ts` | Available | Used for prompt and reference planning. |
| Stable localization-safe names | Creates deterministic fictional names and aliases from locale-neutral pools and reuses the rename map. | `packages/story-localization/src/character-rename.service.ts`, `packages/story-localization/src/full-story-contract.ts` | Available | Prevents original-name leakage in validation. |
| Character reference images | Generates, regenerates, approves, and associates per-character reference images. | `apps/cli/src/index.ts`, `apps/cli/src/episode-commands.ts`, `packages/image-generation/src/episode-image-pipeline.ts` | Available | Human approval state is explicit. |
| Cross-episode character continuity | Reuses shared episode character files and references within an episode workspace. | `apps/cli/src/story-localization-commands.ts`, `apps/cli/src/episode-commands.ts` | Partially available | Cross-episode/franchise registry not verified. |
| Conflict detection | Detects duplicate entity type conflicts, conflicting facts, missing climax/ending, and character name leaks. | `packages/story-localization/src/story-artifact-model.ts`, `packages/story-localization/src/generated-story-validator.ts` | Available | Visual appearance conflict detection not verified. |
| Human review and approval | Prepares review packages and records approve/reject status. | `apps/cli/src/episode-commands.ts`, `packages/dark-truth/src/index.ts` | Available | Used for episode assets and character references. |
| Production readiness analysis | Scores full stories for hook, pacing, clarity, visual suitability, localization, safety, and thumbnail potential. | `packages/story-localization/src/story-production-analysis.ts`, `apps/cli/src/story-analysis-command.ts` | Available | Full stories only in v1. |
| Scene planning | Segments narration into scenes with timing, keywords, visual prompts, and aspect ratios. | `packages/scene-planning/src/index.ts`, `packages/domain/src/index.ts` | Available | One-to-one and beat-based planning. |
| Visual scene specs | Captures focal subject, environment, shot size, camera angle, action, mood, lighting, text requirements, and continuity elements. | `packages/image-generation/src/episode-image-pipeline.ts` | Available | Used in prompt compilation and validation. |
| Shot planning | Generates deterministic shot plans from source scenes, pacing profiles, budgets, crops, transitions, and motion. | `packages/visual-planning/src/shot-planner.ts`, `apps/cli/src/shots.ts` | Available | Supports full and short aspect ratios. |
| Motion effects | Supports crops, pans, pan-and-zoom, push/pull, smart crop, parallax catalog entries, overlays, and transitions. | `packages/domain/src/visual-retention/treatment-catalog.ts`, `packages/rendering/src/index.ts` | Partially available | Some catalog treatments are not rendered; unsupported treatments fail validation/render. |
| Image generation | Generates scene images via OpenAI image APIs with prompt, reference, cache, checkpoint, and failure state. | `packages/image-generation/src/episode-image-pipeline.ts`, `apps/cli/src/index.ts` | Available | Batch and direct paths exist. |
| Image batch processing | Prepares OpenAI image batch manifests, JSONL requests, retry manifests, and import reports. | `packages/image-generation/src/image-batch-planner.ts`, `packages/image-generation/src/image-batch-service.ts` | Available | English full scene image batch schema is explicit. |
| Thumbnail generation | Creates localized full/short thumbnails from reference images, hook text, prompt compilation, and text compositing. | `packages/image-generation/src/story-thumbnail.ts`, `packages/image-generation/src/thumbnail-image-generator.ts`, `apps/cli/src/thumbnail-commands.ts` | Available | Uses OpenAI image edit plus local composition. |
| Narration pipeline | Prepares spoken text, chunks narration, plans performance directions, applies pronunciation transforms, generates TTS, assembles, masters, and validates audio. | `packages/speech/src/narration-pipeline.ts`, `packages/speech/src/performance-direction.ts`, `packages/speech/src/pronunciation.ts` | Available | New pipeline has legacy/shadow/new rollout modes. |
| Voice instructions | Stores language-specific full and short narrator instructions. | `config/voices/dark-truth-documentary/*.txt`, `packages/speech/src/voice-settings.ts` | Available | Current configs cover `en`, `de`, `es`, `fr`; Spanish full/short included. |
| Audio validation and mastering | Checks duration, WPM drift, decodability, silence, sample rate, and loudness-related mastering profiles. | `packages/speech/src/audio-validation.ts`, `packages/speech/src/mastering.ts`, `packages/config/src/index.ts` | Available | Uses ffprobe/ffmpeg where applicable. |
| Captions and synchronization | Builds SRT and caption packs from alignment and transcript timing. | `packages/alignment/src/caption-plan.ts`, `packages/shared/src/index.ts`, `packages/pipeline/src/index.ts` | Available | Timing comes from transcription or narration artifacts. |
| Video rendering | Renders scene clips and final videos with ffmpeg, captions, shot plans, local/remote mode, and output validation. | `packages/rendering/src/index.ts`, `apps/cli/src/index.ts` | Available | Remote rendering can fall back locally. |
| Derived shot cache | Reuses rendered shot clips when fingerprints and dependencies match. | `packages/rendering/src/index.ts`, `packages/rendering/src/derived-shot-cache.unit.test.ts` | Available | Reduces rerendering cost. |
| Metadata generation | Generates YouTube titles, descriptions, chapters, tags, hashtags, thumbnail text, pinned comments, and upload settings. | `packages/metadata/src/youtube-metadata.ts`, `apps/cli/src/index.ts` | Available | TikTok metadata is heuristic only in `packages/metadata/src/index.ts`. |
| YouTube upload | Uploads video, thumbnail, metadata, playlist linkage, and persists upload reports. | `packages/youtube-upload/src/index.ts`, `apps/cli/src/index.ts` | Available | Requires credentials; publishing is a separate boundary. |
| Workflow manifests | Plans multi-locale full/short workflow stages with artifacts, cache status, failures, and batch state. | `packages/story-localization/src/story-workflow-planner.ts`, `packages/story-localization/src/story-workflow.types.ts` | In development | Planning/status wrappers exist; not all media stages are unified under one executor. |
| Durable unified workflow executor | Future state for running all planned story/media stages through one durable DAG. | `docs/plans/story-to-multilocale-batch-pipeline-master-plan.md`, `docs/plans/story-pipeline-implementation-roadmap.md` | Planned | Current implementation has planning/status contracts and subsystem commands, not one completed executor. |
| Manual localized artifact override | Future option to supply a localized story manually with explicit provenance. | `docs/plans/story-to-multilocale-batch-pipeline-master-plan.md` | Planned | The plan says this CLI flag does not exist today. |
| Short production analysis | Future production-analysis extension for short-story quality gates. | `docs/plans/story-pipeline-batch-strategy.md`, `docs/plans/story-to-multilocale-batch-pipeline-master-plan.md` | Planned | Implemented production analysis is full-story only. |
| Media stack consolidation | Future target architecture for one consolidated media stack and extracted workers. | `docs/architecture/target-media-architecture.md`, `docs/migrations/media-consolidation-plan.md` | Planned | Existing media packages are modular but still have legacy compatibility paths. |
| Batch localization | Prepares, submits, refreshes, imports, retries, cancels, verifies, and rebuilds OpenAI batch manifests. | `apps/cli/src/story-localization-commands.ts`, `packages/story-localization/src/story-localization-batch-service.ts` | Available | Supports partial import/failure states. |
| Resume and invalidation | Uses fingerprints, manifests, cache state, source hashes, prompt/schema/model/config fingerprints, and `--resume`. | `packages/story-localization/src/story-workflow-invalidation.ts`, `packages/speech/src/narration-cache.ts`, `packages/rendering/src/index.ts` | Available | Boundaries differ by subsystem. |
| Retry and repair routing | Classifies failures, supports repair model settings, retry failed batches, fallback-to-sync, and terminal failure state. | `packages/story-localization/src/story-retry-routing.ts`, `apps/cli/src/story-localization-commands.ts`, `packages/image-generation/src/image-batch.schemas.ts` | Available | Fallback behavior must be explicitly configured. |
| Observability | Records execution reports, API calls, process executions, generated images, events, retries, aggregate costs, and warnings. | `packages/observability/src/telemetry.ts`, `scripts/run-with-telemetry.mjs` | Available | Pricing is optional. |
| Cost controls | Supports token preflight, cached input accounting, image savings estimates, chunk cache, dry-run, validation-only, and concurrency limits. | `packages/story-localization/src/story-generation-preflight.ts`, `packages/observability/src/pricing.ts`, `packages/observability/src/visual-retention.ts` | Available | Cost estimates require configured pricing for dollar values. |
| Persistence | Uses filesystem artifacts plus SQLite run state. | `packages/persistence/src/index.ts`, `docs/architecture/system-overview.md` | Available | Filesystem remains primary production state. |
| API and web surfaces | Minimal health/static surfaces exist. | `apps/api/src/index.ts`, `apps/web/src/index.ts` | Experimental | CLI is the primary product surface. |
| TikTok publishing | No verified TikTok upload command. | `docs/story-to-video.md`, `apps/cli/src/index.ts` | Not verified | YouTube publishing is implemented. |
