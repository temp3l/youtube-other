# Dark Truth Episodes — Episodes 002–040 Multilingual Production Pack

Included languages:

- English (`en`)
- German (`de`)
- Spanish (`es`)
- French (`fr`)

Every episode folder contains:

- one full episode in each language
- one standalone Short in each language
- localized titles, thumbnail text, descriptions, narration instructions, hashtags and metadata

The corrected English production files from the earlier packs are preserved. The German,
Spanish and French files are complete localized production drafts rather than mixed-language files.

A native-speaker editorial review remains recommended before publication, especially for
regional style, pronunciation and final keyword research.
