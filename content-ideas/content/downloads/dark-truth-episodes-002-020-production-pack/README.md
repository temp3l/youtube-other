# Dark Truth Episodes — Production Pack 002–020

This archive contains corrected production drafts for episodes 002 through 020.

Each episode folder contains:

- `*-full.md`: full narration, sound directions, metadata, disclosure, title and thumbnail recommendation
- `*-short.md`: standalone vertical Short with narration and metadata

## Editorial changes applied

- Removed duplicated `Narration Script` headings and template leakage.
- Removed protagonist-name inconsistencies.
- Replaced the repeated doppelgänger template with topic-specific plots.
- Aligned every story with its title and central legend.
- Added clear fiction, folklore, historical or dramatization disclosures.
- Replaced ineffective one-word tags and oversized thumbnail text.
- Kept violence mostly implied and non-graphic for broader reach and monetization.
- Made Shorts self-contained instead of simple excerpts.
- Kept narration instructions separate from spoken copy.

## Important production note

Episodes based on folklore or internet legends should not be described as verified fact.
The Dyatlov Pass episode must preserve the distinction between documented evidence,
plausible explanations and dramatized reconstruction.
