# Narration Script

Noah vit le clown dans un magasin de jouets verrouillé. Sur la caméra suivante, il attendait déjà
ailleurs. Son sourire s’élargissait lorsque Noah détournait les yeux. Toutes les sorties ramenaient
à l’atrium. Le comportement suivait une règle : La silhouette se déplaçait par les reflets, pas par
les couloirs. Une preuve vérifiable restait pourtant stable. Plus le phénomène était documenté avec
précision, plus il répondait précisément. Puis le son ou le signe central revint d’un endroit
physiquement inaccessible. La pression personnelle augmentait également. Le seul plan possible était
le suivant : Noah éteignit les lumières et recouvrit toutes les surfaces réfléchissantes. Lorsque le
clown apparut dans le sol poli, Noah couvrit le reflet de mousse avant qu’une main ne traverse. Noah
n’eut qu’une seule occasion de suivre le plan. Pendant quelques secondes, tout devint silencieux. Le
clown avait disparu au matin. Sur les photos suivantes, le reflet de Noah portait du maquillage
blanc. L’histoire reste troublante parce que la dernière preuve ne montrait pas une fuite complète.
